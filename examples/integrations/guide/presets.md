# Preset Configs

Some integration options are mutually exclusive at runtime, so this directory keeps them as small copyable presets.

## Algolia Search

Use `examples/integrations/algolia/athen.config.ts` when you want hosted DocSearch instead of the local FlexSearch setup.

## Disabled Analytics

Use `examples/integrations/analytics-disabled/athen.config.ts` when a project should not inject analytics scripts.
