# Integrations Example

This example combines the integration examples into one runnable site.

- Local FlexSearch is enabled in `athen.config.ts`.
- Analytics injects Google Analytics, Plausible, Umami, and a custom snippet.
- A minimal plugin exposes a virtual module and injects HTML tags.
- Algolia and disabled analytics presets live next to the runnable site for copy/paste.
