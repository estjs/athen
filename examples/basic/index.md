# Basic Example

This site merges the small configuration examples into one copyable project.

- Shallow top-level config: `title`, `description`, `lang`, `cleanUrls`, `trailingSlash`, `rewrites`, and `onBrokenLinks`.
- Markdown options: line numbers, page outline depth, external link attributes, and Shiki theme.
- Strict local link checking with a valid page link and anchor link.
