# CLI 命令参考

Athen 提供了一组用于开发、构建和预览的命令行工具（CLI）。

## 基本命令

在安装了 `athen` 的项目中，你可以使用以下命令：

### `athen dev [root]`

启动开发服务器。

- **`root`**: 可选。指定站点的根目录（包含文档文件的目录，如 `docs`）。默认值为当前工作目录 `.`。
- **`--port <port>`**: 可选。指定开发服务器端口，默认值为 `8730`。
- **`--host [host]`**: 可选。指定开发服务器监听的主机。
- **说明**: 该命令会启动一个具有热更新（HMR）支持的本地服务器。

### `athen build [root]`

构建生产环境所需的静态文件。

- **`root`**: 可选。指定站点的根目录。默认值为当前工作目录 `.`。
- **说明**: 该命令会将文档静态生成（SSG）为 HTML，并将其输出到 `dist` 目录中。

### `athen preview [root]`

在本地预览已经构建好的生产环境产物。

- **`root`**: 可选。指定站点的根目录。默认值为当前工作目录 `.`。
- **`--port <port>`**: 可选。指定预览服务器端口。
- **`--host [host]`**: 可选。指定预览服务器监听的主机。
- **说明**: 必须先运行 `athen build`，然后才能通过预览服务器查看打包后的站点。
