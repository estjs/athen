# 国际化

import Locales from '../fragments/locales.mdx';

Athen 提供了一套简单可配置的国际化方案，接下来我们将介绍如何在你的项目中接入。

## 项目结构

我们推荐你将不同语言的文档放在不同的子目录中，例如：

```bash
docs
├── index.md
├── guide
│   └── start.md
├── zh
│   ├── index.md
│   └── guide
│       └── start.md
└── fr
    ├── index.md
    └── guide
        └── start.md
```

> 其中 `index.md` 一般代表 Home 页面。

## locales 配置

在配置文件中添加顶层 `locales` 字段，用于配置不同语言。`locales` 的 key 是语言路由前缀，根语言使用 `/`，其他语言使用 `/zh/`、`/fr/` 这样的前缀：

```ts title="athen.config.ts"
import { defineConfig } from 'athen';

export default defineConfig({
  defaultLocale: 'en',
  locales: {
    '/': {
      label: 'English',
      lang: 'en-US',
      title: 'Athen',
      description: 'Athen is a documentation generator based on Vite & Essor',
      nav: [{ text: 'Guide', link: '/guide/start' }],
      sidebar: {
        '/guide/': [
          {
            text: 'Guide',
            items: [{ text: 'Start', link: '/guide/start' }]
          }
        ]
      }
    },
    '/zh/': {
      label: '简体中文',
      lang: 'zh-CN',
      title: 'Athen',
      description: 'Athen 是一个基于 Vite & Essor 的文档生成工具',
      nav: [{ text: '指南', link: '/zh/guide/start' }],
      sidebar: {
        '/zh/guide/': [
          {
            text: '指南',
            items: [{ text: '开始', link: '/zh/guide/start' }]
          }
        ]
      },
      outlineTitle: '目录',
      prevPageText: '上一页',
      nextPageText: '下一页'
    },
    '/fr/': {
      label: 'Français',
      lang: 'fr-FR',
      title: 'Athen',
      description: 'Athen est un generateur de documentation base sur Vite et Essor',
      nav: [{ text: 'Guide', link: '/fr/guide/start' }],
      sidebar: {
        '/fr/guide/': [
          {
            text: 'Guide',
            items: [{ text: 'Commencer', link: '/fr/guide/start' }]
          }
        ]
      },
      outlineTitle: 'Sur cette page',
      prevPageText: 'Page precedente',
      nextPageText: 'Page suivante'
    }
  }
});
```

当存在多个语言时，默认主题会自动显示语言切换菜单。上面的示例会显示 English、简体中文、Français 三种语言。用户从 `/guide/start` 切换到简体中文时，如果对应翻译页面存在，会进入 `/zh/guide/start`。

也可以在每个语言下直接使用自动侧边栏：

```ts title="athen.config.ts"
export default defineConfig({
  defaultLocale: 'en',
  locales: {
    '/': { label: 'English', lang: 'en-US', sidebar: 'auto' },
    '/zh/': { label: '简体中文', lang: 'zh-CN', sidebar: 'auto' },
    '/fr/': { label: 'Français', lang: 'fr-FR', sidebar: 'auto' }
  }
});
```

其中 `locales` 的 value 为 `LocaleConfig`，通过 `LocaleConfig` 可以配置对应语言的各项参数，其类型定义如下：
