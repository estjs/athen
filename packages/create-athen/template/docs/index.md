## Quick Start

Get up and running in minutes:

```bash
# Create a new project
npm create athen@latest my-docs

# Navigate to project
cd my-docs

# Install dependencies
npm install

# Start development server
npm run dev
```

## Why Athen?

Athen is designed to make documentation simple and powerful. Whether you're building API docs, guides, or knowledge bases, Athen provides the tools you need:

- **Developer-friendly**: Built with modern web technologies
- **Performance-focused**: Optimized for speed and SEO
- **Feature-rich**: Search, i18n, theming, and more out of the box
- **Extensible**: Plugin system for custom functionality
