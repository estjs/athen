# 介绍

欢迎使用 Athen！本指南将帮助您开始构建美观的文档站点。

## 什么是 Athen？

Athen 是一个专为文档设计的现代静态站点生成器。它将 Markdown 的简洁性与现代 Web 技术的强大功能相结合，创建快速、美观且功能丰富的文档站点。

## 主要特性

### 🚀 **性能优先**
- 静态站点生成，实现最佳加载速度
- 优化的包分割和懒加载
- 内置性能监控

### 📝 **Markdown & MDX 支持**
- 使用熟悉的 Markdown 语法编写内容
- 使用 MDX 创建交互式组件
- 代码块语法高亮
- 使用 KaTeX 支持数学表达式

### 🔍 **强大搜索**
- 内置全文搜索
- 支持多语言
- 即时搜索结果
- 键盘快捷键

### 🌍 **国际化**
- 开箱即用的多语言支持
- 自动语言检测
- 本地化导航和内容

### 🎨 **美观主题**
- 现代响应式设计
- 深色和浅色模式支持
- 可定制的配色方案
- 移动优先的设计方法
