# 更新日志

此项目的所有重要更改都将记录在此文件中。

## [1.0.0] - 2024-01-10

### 新增
- 初始版本发布
- 基础文档结构
- FlexSearch 搜索功能
- 多语言支持（英文/中文）
- 响应式设计与深色模式
- 示例页面和 API 文档

### 功能特性
- 全文档全文搜索
- 自动语言检测的国际化
- 移动响应式设计
- 深色/浅色主题切换
- 社交媒体集成
- SEO 优化

## 如何更新

更新到最新版本：

```bash
npm update athen @athen/plugin-search
```
