## 快速开始

几分钟内即可启动运行：

```bash
# 创建新项目
npm create athen@latest my-docs

# 进入项目目录
cd my-docs

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

## 为什么选择 Athen？

Athen 旨在让文档变得简单而强大。无论您是构建 API 文档、指南还是知识库，Athen 都能提供您需要的工具：

- **开发者友好**: 使用现代 Web 技术构建
- **性能优先**: 针对速度和 SEO 进行优化
- **功能丰富**: 开箱即用的搜索、国际化、主题等功能
- **可扩展**: 插件系统支持自定义功能
